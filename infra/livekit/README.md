# Local LiveKit Stack

This folder runs a local LiveKit stack for development:

- `livekit-server`
- `redis`
- `livekit/egress`

## Start

```bash
./infra/livekit/up.sh
```

## Use with this project

```bash
source infra/livekit/use-local-env.sh
```

## Logs

```bash
./infra/livekit/logs.sh
```

## Stop

```bash
./infra/livekit/down.sh
```
